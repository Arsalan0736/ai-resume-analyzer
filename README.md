## ‚ú® Features

- **AI-Powered Analysis**: Get detailed insights into your resume's strengths and weaknesses.
- **ATS Compatibility**: Check if your resume is optimized for Applicant Tracking Systems.
- **Instant Feedback**: Receive real-time suggestions to improve your resume's impact.
- **PDF Support**: Seamlessly parse and analyze PDF resumes using `pdfjs-dist`.
- **Modern UI**: A clean, responsive, and beautiful interface built with Tailwind CSS.
- **Secure & Fast**: Client-side processing ensures your data stays private and analysis is lightning fast.

## ‚ö° Getting Started

Follow these steps to set up the project locally on your machine.

### Prerequisites

Make sure you have the following installed:
- [Node.js](https://nodejs.org/) (v18 or higher)
- [npm](https://www.npmjs.com/) (usually comes with Node.js)

### Installation

1.  **Clone the repository**
    ```bash
    git clone https://github.com/Arsalan0736/ai-resume-analyzer.git
    cd ai-resume-analyzer
    ```

2.  **Install dependencies**
    ```bash
    npm install
    ```

3.  **Start the development server**
    ```bash
    npm run dev
    ```

4.  **Open your browser**
    Navigate to `http://localhost:5173` to see the application running.

## üì¶ Build for Production
